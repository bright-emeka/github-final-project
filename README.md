# github-final-project

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

## Inputs
- p: principal amount
- t: time period in years
- r: annual rate of interest

## Output
simple interest = p * t * r / 100

## Getting Started
To get started, clone the repository and run the `simple-interest.sh` script. Follow the prompts to enter the principal amount, annual rate of interest, and time period in years. The script will calculate and display the simple interest.
